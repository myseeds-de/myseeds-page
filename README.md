# MySeeds GitHub Pages

A lightweight public landing page for [MySeeds.de](https://myseeds.de/).

## Publish with GitHub Pages

1. Create a public repository named `myseeds-page`.
2. Upload all files from this package to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
5. Save the settings.

The expected public URL is:

`https://jvb94.github.io/myseeds-page/`

## Files

- `index.html` – page structure and content
- `style.css` – responsive MySeeds styling
- `favicon.svg` – simple seed favicon
- `robots.txt` – crawler rules
- `sitemap.xml` – one-page sitemap
- `.nojekyll` – disables Jekyll processing

## Main website

The canonical URL points to `https://myseeds.de/` to avoid competing with the main shop in search results.
